# Image Format Converter — Pro (static build)

This folder is fully hostable on any static server. Just open `index.html` locally or upload to GitHub Pages/Netlify/Vercel/S3/Cloudflare Pages.

- No build step — uses ESM CDNs
- All processing is client-side (privacy-friendly)
- EXIF passthrough supported for JPEG→JPEG (best-effort)

## Files
- index.html — the entire app
